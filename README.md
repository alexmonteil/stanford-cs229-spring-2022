## 📚 How do I take the course?

Follow these steps to self-study the course material effectively:

1. **Download the repository:** Clone or download the content of this repository to your local machine.
2. **Consult the schedule:** Refer to `course_schedule.pdf` to see exactly what topics and materials are covered in each lecture block.
3. **Watch the Lectures:** Follow along visually with the full video lectures provided in the [YouTube playlist](https://youtube.com/playlist?list=PLoROMvodv4rNyWOpJg_Yh4NSqI4Z4vOYy&si=0EsyajdQuHIOjURQ).
4. **Read the Main Notes:** Dive deep into the specific section of the core `main_notes.pdf` file mapped to that lecture.
5. **Check for Additional Reading:** Some lectures (but not all) cover supplementary concepts. When indicated by the schedule, check the folder named `additional notes` for extra material.
6. **Complete Problem Sets:** Test your understanding and apply the mathematical concepts by tackling the assignments located in the `problem sets/` folder as you progress.
